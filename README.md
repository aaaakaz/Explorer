# Explorer

A place discovery website where users can search, browse and review locations around the world.

Built for 6CS028 Advanced Web Technologies at the University of Wolverhampton.

## Tech Stack

- CodeIgniter 4 (PHP)
- MySQL
- Bootstrap 5
- Google Maps API
- OpenWeatherMap API
- Unsplash API

## Features

- Browse and search places by category
- User registration and login
- Leave reviews and star ratings
- Live weather for each location
- Interactive Google Maps embed
- Dark mode toggle
- Responsive design

## Setup

Hosted on mi-linux.wlv.ac.uk. Database config in `.env` (not included in repo).
